# Node.js API

This is a Node.js API project that uses various packages for functionality. Follow the instructions below to set up the project on your local machine.

## Prerequisites

Before running the project, ensure that the following is installed on your local machine:

- **Node.js**: You can download and install Node.js from the [official website](https://nodejs.org/).

## Setup

Once Node.js is installed, follow these steps to set up the project:

1. Clone the repository to your local machine.
   
   ```bash
   git clone <repository-url>
   cd <repository-directory>

2. Initialize a new Node.js project by running the following command:

	```bash
	npm init -y

3. Install the necessary dependencies:

	```bash
	npm i express bcrypt cors dotenv jsonwebtoken pg

4. Install nodemon for development (recommended):

	```bash	
	npm i -D nodemon

 ## Running the API
 
To start the server, you can run the following command:

  ```bash
	npm start
